210×90 Grid Image Editor

Edit traced grid images without moving the canvas workspace.
Grid: 210 columns × 90 rows. Transparent cells: 18,900.
Underlay: none
Grid import: none
Cell: —
100%

Click or drag with Brush to paint cells. Bucket fill replaces a contiguous region of matching cells. Line draws a straight grid line from press to release, with live preview. Imported grid PNGs can be the editor size or any integer export scale of it, and visible import pixels are rounded to black or white automatically. Grid image opacity affects the editor preview only, so exports stay strict black, white, and transparent. Use the canvas zoom buttons to inspect details. Shortcuts: 1 Cycle, 2 Transparent, 3 White, 4 Black, B Brush, F Fill, L Line, Ctrl/⌘+Z Undo, Ctrl/⌘+Y or Ctrl/⌘+Shift+Z Redo, Ctrl/⌘+S Export.
